# Decisiones Tecnológicas

La selección de las tecnologías utilizadas en el desarrollo del frontend se fundamenta en criterios de robustez, mantenibilidad y pertinencia académica.

- React: Permite el desarrollo de interfaces basadas en componentes reutilizables.
- TypeScript: Aporta tipado estático, reduciendo errores en tiempo de desarrollo.
- Vite: Proporciona un entorno de desarrollo rápido y eficiente.
- Git y GitHub: Facilitan el control de versiones y la trazabilidad del desarrollo.

Estas tecnologías están alineadas con buenas prácticas de la ingeniería de software y resultan adecuadas para un proyecto de grado de carácter académico.
