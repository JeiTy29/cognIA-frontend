# Alcance del Frontend

El alcance del frontend del sistema se limita al diseño, implementación y documentación de la interfaz gráfica de usuario del aplicativo web. En esta capa no se implementan directamente los algoritmos de inteligencia artificial ni la lógica de negocio asociada al procesamiento de datos, los cuales serán responsabilidad del backend del sistema.

Las funcionalidades contempladas dentro del alcance del frontend incluyen:

- Visualización de una página principal de acceso público con información general del sistema.
- Presentación de interfaces diferenciadas según el rol del usuario (padre o tutor, psicólogo y administrador).
- Recolección de información mediante cuestionarios estructurados.
- Visualización de resultados y evaluaciones emitidas por el sistema.
- Preparación de la estructura necesaria para la integración futura con los servicios del backend.
