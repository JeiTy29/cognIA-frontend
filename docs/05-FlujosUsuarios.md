# Flujo de Usuario

El sistema contempla distintos tipos de usuarios, cada uno con un flujo de interacción específico dentro del aplicativo web.

## Padre o Tutor

1. Accede a la página principal del sistema.
2. Inicia sesión en la plataforma.
3. Diligencia el cuestionario psicológico correspondiente al menor.
4. Visualiza los resultados de la evaluación generada por el sistema.

## Psicólogo

1. Inicia sesión en el sistema.
2. Accede a un panel de control con las evaluaciones disponibles.
3. Analiza los resultados obtenidos a partir de los cuestionarios.

## Administrador

1. Inicia sesión con privilegios administrativos.
2. Gestiona usuarios y supervisa el funcionamiento general del sistema.

Estos flujos permiten definir claramente los casos de uso del sistema, los cuales servirán como base para el análisis funcional del proyecto.
