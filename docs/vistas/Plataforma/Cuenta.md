# Vista: Cuenta

## Propósito

Gestionar información del perfil del usuario según el rol.

## Roles

**Padre/Tutor**
- Perfil básico y preferencias.

**Psicólogo**
- Datos profesionales y permisos.

## Navegación

- Desde Sidebar → Cuenta.
- Rutas: `/padre/cuenta` y `/psicologo/cuenta`.

## Componentes futuros esperados

- Edición de perfil.
- Seguridad (cambio de contraseña).
- Preferencias y notificaciones.
